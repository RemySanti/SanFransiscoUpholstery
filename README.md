# SFUG Remake

New repository for rebuilding [San Francisco Upholstery Group](https://sanfranciscoupholsterygroup.com/) with a modern, conversion-focused marketing site.

## Stack

- React
- TypeScript
- Vite

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Current status

- Project scaffolded in a new git repo
- Premium homepage structure in place:
  - Hero + conversion CTA
  - Services grid
  - About + stats
  - Projects/verticals section
  - FAQ
  - Contact section
